Project Overview
This project presents a Smart Embedded Neonatal Incubator Monitoring and Environmental Control System designed to maintain safe conditions for newborn care. The system monitors important parameters like temperature and humidity inside the incubator and helps maintain a stable environment. It also provides alerts when the temperature exceeds 35°C to ensure infant safety.

Technologies Used
Arduino / Microcontroller
Temperature and Humidity Sensors
Embedded System Design
Basic IoT Monitoring
Arduino IDE for programming

Working Principle
Sensors placed inside the incubator continuously measure temperature and humidity. The microcontroller processes this data and displays the readings. If the temperature exceeds the safe limit (35°C), the system triggers an alert/notification to indicate unsafe conditions, helping maintain a controlled environment for newborns.


Future Improvements
Add mobile or IoT-based remote monitoring

Implement data logging for long-term monitoring
